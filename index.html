GBIF Static map

Displays static maps with occurence data for species using the GBIF apis.

Examples

Example 1: With automatic caption text.

Code: <figure class="gbif-static-map" data-zoom-level="5" data-top-left="9,20" data-bottom-right="10,21" data-taxon-key="2669001" data-points-style="green2.poly" data-tiles-style="gbif-middle" aria-label="Map showing Magellans peat moss observations in Argentina and Chile."> <figcaption class="caption auto-caption">This text will be overridden</figcaption> </figure>

Example 2: With manual caption text.

Code: <figure class="gbif-static-map" data-zoom-level="8" data-top-left="134,68" data-bottom-right="136,69" data-taxon-key="9805720" data-points-style="purpleHeat.point" data-tiles-style="osm-bright-en" aria-label="Map showing magpie observations in middle of Norway."> <figcaption class="caption">This text will not be overridden</figcaption> </figure>

Parameters

Parameters are set as data- attributes on the .gbif-static-map element.

Parameter Values
data-zoom-level

Integer, 0-28.

Level 0 equals the whole world in two tiles, level 5 means Iceland fit into one tile, level 11 one tile fits Manhattan, level 20 one tile roughly equals the foundation of one the the feet of the Eiffel Tower.
data-top

Top boundary of the map. Integer, a product of the zoom level.

See https://www.maptiler.com/google-maps-coordinates-tile-bounds-projection/ for more info.
data-right

Right boundary. Integer, a product of the zoom level.
data-bottom

Bottom boundary. Integer, a product of the zoom level.
data-left

Left boundary. Integer, a product of the zoom level.
data-top-left

Short form of setting the top and left boundary values.
data-bottom-right

Short form of setting the bottom and right boundary values.
data-taxon-key

Integer, a valid GBIF taxon id.

See https://www.gbif.org/species/search for more details.
data-points-style

String. Name of a valid colour style. Defaults to classic.point.

See https://www.gbif.org/developer/maps#colors for a list of the colours.
data-tiles-style

String. A valid raster style for the background tiles. Defaults to gbif-classic.

See https://tile.gbif.org/ui/ for a examples of valid styles.

Auto-caption

Add an element with the classes "caption auto-caption" to the .gbif-static-map element.

If you aren't using the auto-caption, it is strongly adviced to add an link to teh GBIF data in some other way.
